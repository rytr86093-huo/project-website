## 💻 Функционал проекта
1. **Каталог товаров (`qqq.html`)** — главная страница с распределением цветочной продукции по карточкам (изображение, описание, цена и кнопка добавления).
2. **Интерактивная корзина (`qqq2.html`)** — динамический интерфейс с добавлением/удалением товаров, автоматическим расчетом стоимости заказа и стоимости доставки, а также блокировкой оформления при пустой корзине.
3. **Сохранение данных (`localStorage`)** — состояние корзины покупок стабильно сохраняется между сессиями и не пропадает при перезагрузке страниц.
4. **Страница «О нас» (`qqq3.html`)** — информационный блок с интерактивным слайдером (каруселью) изображений, управляемым кнопками.
5. **Адаптивный дизайн** — верстка корректно отображается на любых устройствах с шириной экрана от 320px до 1920px.

## 🛠 Технологический стек
Проект реализован строго на базовых frontend-технологиях без использования сторонних фреймворков и библиотек (чистый код):
* **HTML5** — семантическая разметка страниц
* **CSS3** — стилизация, сетки Flexbox/Grid, медиазапросы (`@media`)
* **JavaScript (ES6+)** — клиентская логика корзины, слайдера и интеграция с локальным хранилищем браузера

## 🚀 Инструкция по запуску и развертыванию
### Локальный запуск (Оффлайн-демонстрация)
1. Скачайте файлы проекта или клонируйте данный репозиторий.
2. Откройте файл `qqq.html` в любом современном веб-браузере (Google Chrome, Yandex Browser, Mozilla Firefox, Microsoft Edge) двойным щелчком мыши.

### Требования к клиентскому оборудованию
* Оперативная память: от 2 ГБ
* Монитор: разрешение от 1024×768 пикселей (рекомендуется 1920×1080)
* Наличие мыши, клавиатуры или сенсорного управления
